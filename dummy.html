This is a list of HTML5 elements, sometimes several in various ways. Some may not appear, depending on your browser.

Some elements may appear to be incorrect due to useragent stylesheets which vary from browser to browser.

Tags are placed within semantic elements in the following format:

  • Main Categories [article]
    • Title [header]
    • Element Lists [sections]
    • Link to top [footer]

This may cause some elements to appear incorrectly - for example, in Chrome, H1 tags within an Article are smaller.

I'll try to update with tags as they're commonly used together.
